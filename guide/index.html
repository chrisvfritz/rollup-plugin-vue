# Getting Started

Add rollup-plugin-vue to plugins option of the rollup config. That's pretty much everything.



 





 




// file: rollup.config.js

import vue from 'rollup-plugin-vue'

export default {
  // ...
  plugins: [
    // ...
    vue(/* options */)
    // ...
  ]
}
